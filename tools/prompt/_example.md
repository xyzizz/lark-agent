# 提示词模板说明

文件名即为模板类型（如 `issue.md` 对应 `issue` 类型），支持 Go text/template 语法。

## 可用变量

### 通用变量（所有模板可用）

| 变量 | 说明 |
|------|------|
| `{{.Message}}` | 用户发送的消息原文 |

### intent.md 专用

| 变量 | 说明 |
|------|------|
| `{{.ProjectList}}` | 当前已启用的项目路由列表（名称 + 关键词），供 LLM 匹配 suspected_project |

### issue.md / requirement.md 共用

| 变量 | 说明 |
|------|------|
| `{{.ProjectName}}` | 路由匹配到的项目名称 |
| `{{.Repos}}` | 项目关联的仓库列表（格式：`- 仓库: /path — 描述`） |

### requirement.md 专用

| 变量 | 说明 |
|------|------|
| `{{.DocContent}}` | 从路由配置的飞书文档 URL 拉取到的文档内容 |

## 条件渲染

```
{{if .ProjectName}}
有项目信息时才显示这段
{{end}}
```

## 注意事项

- 文件名以下划线开头（如本文件）不会被加载
- system.md 不使用变量，直接写纯文本即可
- 修改文件后立即生效，无需重启服务
